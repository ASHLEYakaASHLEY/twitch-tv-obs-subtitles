Twitch Subtiles
Add subtitles to your stream, followers read what you say.
1 Enable your microphone.
2 Complete this form and Come back to this page! You need free API keys. When you complete the form, this page will generate the URL in step 3 with your API Keys for your OBS browser source. If you already have API Keys you can login here instead.
3 Copy URL into OBS Browser Source. Watch Tutorial Video.
4 Customize the look: You may fork the GitHub repo so you can remove watermarks and update colors. Read further details below on this page for free.
5 Keep this page open while you stream. This page captures your voice transcript and transmits the subtitles into OBS via PubNub.
